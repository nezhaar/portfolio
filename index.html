🤖 The Kraken – Bot Discord

The Kraken est un bot Discord développé en JavaScript avec Discord.js. Il permet la modération automatique, la gestion de musique via YouTube, et l’intégration d’API modernes comme GPT et MongoDB.

  • 🛡️ Commandes de modération : kick, ban, clear, mute
  • 🎵 Commandes musicales : !play, !skip, !stop
  • 🧠 Base de données MongoDB + système de logs
